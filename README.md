# Vasu-Food

A simple food ordering web application with a modern UI.

## Features

- Browse food items
- Add to cart
- Checkout and order
- User authentication
- Admin dashboard

## Tech Stack

- React
- Node.js
- Express
- MongoDB
- CSS

## Getting Started

1. **Install dependencies**
   ```fish
   npm install
   ```
2. **Start the development server**
   ```fish
   npm start
   ```

## Folder Structure

- `src/` - Main source code
- `public/` - Static assets
- `package.json` - Project configuration
